A bit of Internals of Git

References
  • branch, remote branch, tag

    都是指向object的指针(一般都指向commit object) 都存储在.git/refs/.

  • Symbolic References

    一种特殊的ref,不直接指向object，而是指向其他的ref，比如HEAD，指向current branch。存储在.git/HEAD
